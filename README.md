# 🎨 Orthodox Iconography & Data Science: The Samokov School Case
**Digital Analysis and Enhancement of Bulgarian National Revival Art**

---

## 🌟 Overview
This project represents an interdisciplinary bridge between **Art History** and **Modern Data Science**. It focuses on the **Samokov Iconographic School**—the most prominent artistic movement during the Bulgarian National Revival. The goal is to extract new knowledge from historical data and provide digital tools for the preservation of darkened or damaged religious art.

## 🎯 Objectives
* **Data Synthesis:** Merging traditional art historical archives with digital datasets.
* **Comparative Analysis:** Placing Bulgarian iconography within the broader context of Eastern Orthodox and Western Christian art.
* **Digital Restoration:** Implementing Computer Vision algorithms (CLAHE) to reveal hidden details in centuries-old icons.

## 🛠️ Tools & Technologies
* **Python 3.x**
* **Data Analysis:** `Pandas`, `NumPy`
* **Visualization:** `Matplotlib`, `Seaborn`
* **Computer Vision:** `OpenCV` (Open Source Computer Vision Library)

## 📂 Repository Structure
* `Pantokrator_vs_Maiestas.ipynb` - The main research notebook containing all code and analysis.
* `icon.jpg` - Sample high-resolution image used for CLAHE demonstration.
* `README.md` - Project documentation and metadata.

## 🖼️ Key Feature: Digital Enhancement
Using **Contrast Limited Adaptive Histogram Equalization (CLAHE)**, this project demonstrates how we can "pierce" through layers of darkened varnish and soot to recover the original mastery of the Samokov painters without touching the physical artifact.
